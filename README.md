## Getting Started

* Clone and init working directory

        $ git clone git@github.com:cristianferrarig/bootstrap-more.git
        $ cd bootstrap-more

* Install middleman gem:

        $ gem install middleman
        $ bundle install
        $ bundle exec middleman server
